# SmartCV Extractor

AI-powered Resume Parsing System built using React, FastAPI, NLP, and MongoDB.

## Features

- Upload PDF/DOCX resumes
- Extract:
  - Name
  - Email
  - Phone
  - Skills
  - Projects
  - Qualification
  - Experience
- Resume dashboard UI
- MongoDB integration
- FastAPI backend
- React frontend

## Tech Stack

Frontend:
- React
- Vite
- TailwindCSS

Backend:
- FastAPI
- Python
- spaCy / NLP
- PyPDF2
- MongoDB

## Run Frontend

```bash
cd frontend
npm install
